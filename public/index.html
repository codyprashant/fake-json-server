JSON Server

Hi, I am here availble to serve you JSON server of all countries. You can use by using below mentioned filtered endpoints for your usage. If you found any error or requires new feature please raise an issue on my github account.

Get all Countries with their various details like Name, Capital, Region, and Languages :
https://json-countries-api.herokuapp.com/countries
Get Country record filtered with their Name :
https://json-countries-api.herokuapp.com/countries?name=India
Get multiple Countries filtered with their Name :
https://json-countries-api.herokuapp.com/countries?name=India&name=Sudan
Get Country record filtered with their region :
https://json-countries-api.herokuapp.com/countries?region=Asia
Get Country record filtered with their region and name :
https://json-countries-api.herokuapp.com/countries?region=Africa&name=Sudan
Get all Countries using pagination :
https://json-countries-api.herokuapp.com/countries?_page=1&_limit=2
Get all Countries record by sorting name :
https://json-countries-api.herokuapp.com/countries?_sort=name&_order=asc
Get the Country record by search the string in records:
https://json-countries-api.herokuapp.com/countries?q=Afghanistan
